# 打工人发泄网页

一个让打工人释放压力的互动网页，疯狂敲击键盘打碎"加班"！

## 功能特点

- 🎯 巨大的"加班"文字居中显示
- ⌨️ 疯狂敲击键盘任意键触发特效
- 💥 敲击15次后"加班"文字会爆炸破碎
- ✨ 每次敲击弹出"拒绝"、"不干了"、"我要下班"等漂浮文字
- 🎨 炫酷的渐变背景和发光特效
- 🔄 可以重置重新开始

## 技术栈

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## 安装依赖

```bash
npm install
# 或
yarn install
# 或
pnpm install
```

## 运行项目

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看效果。

## 使用说明

1. 打开网页后，你会看到巨大的"加班"两个字
2. 疯狂敲击键盘上的任意键
3. 每次敲击都会出现漂浮的发泄文字
4. 敲击15次后，"加班"会爆炸破碎成粒子
5. 点击"再来一次"按钮可以重新开始

## 构建生产版本

```bash
npm run build
npm start
```
